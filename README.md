# COMP2068 Node Flix Server

Node / Express REST API Backend

## NPM Commands

`npm init -y` - initialize npm

`npm i typescript ts-node express` - use typescript & express

`npm run dev` - start in development mode locally

`npm i body-parser` - so express can read request body on POST / PUT

## Lesson 5 Notes & NPM Commands
`npm i mongoose` - for MongoDB connection & CRUD access

## Lesson 6 Notes & NPM Commands
`npm i swagger-jsdoc swagger-ui-express` - swagger API doc generator
